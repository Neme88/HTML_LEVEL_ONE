Form example in Html

form is a very crucial part of html, form plays the primary role of a vehicle which allow interaction between the user and our server.

Interactions like login, getting feedbacks or allowing to submit their contact info for us to reach out to them or to fill out application Etc.

Their are a lots of moving parts that makes a form work effectively which I will list some below

.
  1. form tag: The form tag is used to create a form element. The form tag has a action attribute that specifies the URL where the form data should be sent when the form is submitted. The method attribute specifies the HTTP method (GET or POST) to use for sending the form data.
  2. input tag: The input tag is used to create form controls like text fields, checkboxes, radio buttons, submit buttons, reset buttons, and more. The type attribute specifies the type of input field to create. Some common types include text, email, password, number, checkbox, radio, submit, reset, and file.
  3. label tag: The label tag is used to define a caption for a form control. The for attribute specifies the id of the form control the label is associated with.
    4. button tag: The button tag is used to create interactive buttons in a form. The type attribute specifies the type of button (submit, reset, button). The value attribute specifies the button's displayed text.
  4. select tag: The select tag is used to create a drop-down list. The option tag is used to create individual options in the drop-down list.
  5. textarea tag: The textarea tag is used to create a box to fit in more text.
  6. fieldset tag: The fieldset tag is used to group related form elements. The legend tag is used to create a caption for the fieldset.
